# Dominos_pizza_sales---SQL
🍕 Domino’s Pizza Sales Analysis (SQL Project)

This project analyzes Domino’s Pizza sales data using SQL to extract meaningful business insights related to orders, revenue, and customer behavior.

🔹 Project Overview
Designed and created a relational database (Dominos) with tables for orders and order details
Performed data analysis using SQL queries to understand sales performance and trends
Extracted insights on revenue, top-selling pizzas, order patterns, and peak hours
🔹 Key Analysis Performed
Total number of orders placed
Total revenue generated from pizza sales
Highest-priced pizza and most common pizza size
Top 5 most ordered pizza types
Category-wise pizza sales distribution
Hourly order distribution (peak hours analysis)
Average pizzas ordered per day
Top 3 pizzas by revenue
Revenue contribution by category
Cumulative revenue analysis over time
🔹 Tools & Technologies
SQL (MySQL)
Database Design & Query Optimization
🔹 Key Insights
Identified best-selling pizzas and high-revenue contributors
Discovered peak order hours and daily sales trends
Analyzed category performance to support business decisions
